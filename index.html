Associe a seguir qual verbo HTTP corresponde com a afirmativa:

GET: Verbo HTTP responsável por capturar dados muito utilizado para listar dados vindo do backend;

PATCH: Verbo HTTP utilizado para alterar apenas um de seus elementos normalmente envia-se um parâmetro pela URL;

PUT: Verbo HTTP utilizado para alterar todos os seus elementos;

POST: Verbo HTTP utilizado para enviar dados no corpo da requisição, responsável muitas vezes pelo cadastro de novos itens;
